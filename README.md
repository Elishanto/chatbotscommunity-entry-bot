# @CBCEntryBot
> Telegram bot

## Install dependencies

Install with pip

```sh
$ pip install -r requirements.txt
```

Rename config.example.yml to config.yml

```sh
$ mv config.example.yml config.yml
```

Add API keys and choose db in config.yml
```
TELEGRAM_API_TOKEN: TELEGRAM_API_TOKEN
db: redis OR list
```
